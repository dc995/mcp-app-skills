# VS Code MCP Apps validation — June 2026

**Evidence type:** empirical

Validated in VS Code Insiders using first-party MCP App probes covering:

- Dynamic code execution and external resource CSP behavior
- External UI fetch and media loading
- Microphone, camera and geolocation grants
- Popup behavior
- MCP Apps handshake timing
- `sampling/createMessage` over stateful Streamable HTTP

The detailed capability values are recorded in
`mcp-app-hosts/host-matrix.json`. These observations describe the tested VS Code
Insiders builds; they are not official promises for future builds.

Revalidation must record the VS Code build, MCP extension/runtime build, browser
engine, OS, MCP SDK version and MCP Apps SDK version.
